# ENCICLOPEDIA HIP HOP -- ETAPA 1 (MVP)

## Objetivo

Construir la primera base de conocimiento para **Sopa de Knowledge**,
con aproximadamente **900 conceptos** organizados por categorías. Esta
etapa permitirá ofrecer cientos de niveles, desafíos y una progresión
sólida desde el primer lanzamiento.

# Estructura de cada concepto

Cada entrada de la enciclopedia deberá incluir:

-   Nombre
-   Categoría
-   Definición breve
-   Explicación ampliada
-   Importancia dentro de la cultura
-   Palabras relacionadas
-   Nivel de dificultad
-   Etiquetas
-   Imagen ilustrativa (opcional)
-   Estado (pendiente / revisado / publicado)

------------------------------------------------------------------------

# Objetivo de esta etapa

La Etapa 1 debe proporcionar suficiente contenido para construir una
versión inicial del juego con cientos de niveles, una colección
significativa y una base escalable que permita seguir incorporando
conocimiento sin modificar la arquitectura del proyecto.
